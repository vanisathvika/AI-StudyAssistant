# 📚 AI Study Assistant

An AI-powered Study Assistant built using **Python**, **Google Gemini API**, and **Gradio**.

## 🚀 Features

- 🤖 Ask any study-related question
- 📖 AI explains concepts in simple language
- 💡 Gives examples whenever possible
- ⚡ Fast responses using Gemini AI
- 🎨 Simple and interactive web interface

---

## 🛠 Tech Stack

- Python
- Google Gemini API
- Gradio
- python-dotenv

## ⚙ Installation

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run

```bash
python app.py
```
